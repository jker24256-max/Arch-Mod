# JKER OS // Desktop Keyboard Shortcuts

This document lists the keyboard shortcuts configured for the JKER OS Hyprland environment.

---

## 1. System Applications & Menus

| Key Combination | Action Description | Target Executable |
| :--- | :--- | :--- |
| `Super + Q` | Launch default terminal emulator | Kitty |
| `Super + E` | Open graphical file manager | Thunar |
| `Super + R` | Launch application menu searches | Rofi Launcher |
| `Super + C` | Close active focused window | Hyprland Kill |
| `Super + M` | Exit active Hyprland graphical session | Hyprland Quit |

---

## 2. JKER Custom Applications

| Key Combination | Action Description | Target Executable |
| :--- | :--- | :--- |
| `Super + B` | Launch BlackVault Secure Password Manager | `/usr/bin/blackvault` |
| `Super + S` | Launch SentryOps Network Threat Scanner | `/usr/bin/sentryops` |
| `Super + D` | Launch Download Center Store | `/usr/bin/downloadcenter` |
| `Super + P` | Launch JKER Control Center settings | `/usr/bin/controlcenter` |

---

## 3. Window & Focus Management

* **Move Focus**: `Super + [Left / Right / Up / Down]` arrow keys.
* **Toggle Floating**: `Super + V` (forces target window to float).
* **Switch Workspaces**: `Super + [1 - 5]` (cycles workspaces 1 to 5).
* **Move Window to Workspace**: `Super + Shift + [1 - 5]`.

---

## 4. Hardware Media Keys

* **Raise Volume**: `AudioRaiseVolume` (Fn key) -> Increases volume by 5% via WirePlumber.
* **Lower Volume**: `AudioLowerVolume` (Fn key) -> Decreases volume by 5% via WirePlumber.
* **Mute Audio**: `AudioMute` (Fn key) -> Toggles audio mute state.
* **Increase Brightness**: `MonBrightnessUp` (Fn key) -> Increases screen backlights.
* **Decrease Brightness**: `MonBrightnessDown` (Fn key) -> Decreases screen backlights.
